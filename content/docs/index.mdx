# Welcome to Better Loyalty

This is the documentation for the Better Loyalty platform.

## Getting Started

Follow these steps to get started with Better Loyalty.

### Installation

Install the package using npm:

```bash
npm install better-loyalty
```

### Usage

Basic usage example:

```javascript
import { BetterLoyalty } from "better-loyalty";

const loyalty = new BetterLoyalty();
```
